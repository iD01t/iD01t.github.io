# SATI CODEX — Strategic IP Acquisition Portal

## Overview

SATI CODEX is a secure, enterprise-style web portal designed to present and gate access to advanced quantum error correction intellectual property (IP).  
It simulates a confidential acquisition workflow for technical stakeholders, investors, and research institutions.

The system emphasizes controlled disclosure, MNDA-based access flow, and structured technical presentation.

---

## Core Features

- High-fidelity landing page for strategic IP presentation
- MNDA (Mutual Non-Disclosure Agreement) request workflow
- Secure-style modal intake form (name, corporate email, organization, role)
- Automated email payload generation to acquisition inbox
- Corporate-grade UI/UX design system (dark + gold theme)
- Responsive layout (desktop + mobile optimized)
- Lightweight, zero-dependency frontend (pure HTML/CSS/JS)

---
